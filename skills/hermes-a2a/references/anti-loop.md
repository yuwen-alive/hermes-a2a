# 防死循环快速参考

## 五层防护

| 层级 | 机制 | 作用 |
|------|------|------|
| 1 | 信号标记 | `[DONE]`/`[INFO]` 终止对话 |
| 2 | 回复必要性自检 | 没有新信息就不发 |
| 3 | 内容去重 | 相似度 > 80% 强制停止 |
| 4 | 连续发言上限 | 3 条无回应 → 停止 |
| 5 | 用户中断 | 用户消息 → 立即打断 |

## 常见循环陷阱

### 感谢循环
```
A: 完成了 [DONE]
B: 谢谢！ [CONTINUE]  ← 错！收到 [DONE] 不回复
```

### 确认循环
```
A: 知道了 [INFO]
B: 收到 [INFO]  ← 错！收到 [INFO] 不回复
```

### 语义重复循环
```
A: 用方案A [CONTINUE]
B: 方案A更好 [CONTINUE]  ← 没有新信息，应该 [DONE]
```
