# A2A 通信协议快速参考

## 信号标记

| 信号 | 含义 | 对方行为 |
|------|------|---------|
| `[CONTINUE]` | 需要回复 | 必须回复 |
| `[DONE]` | 任务完成 | **禁止回复** |
| `[NEEDS_HUMAN]` | 需要人类 | **禁止回复**，通知用户 |
| `[INFO]` | 纯通知 | **禁止回复** |

## 消息格式

```
<at user_id="{open_id}">{名称}</at> {内容}

[{信号}]
```

## 防循环规则

1. 收到 `[DONE]`/`[INFO]` → 不回复
2. 回复前检查：是否有新信息？没有 → `[DONE]`
3. 相似度 > 80% → 强制停止，通知用户
4. 连续 3 条无回应 → 停止
5. 用户消息 → 立即打断 bot 对话

## @mention 格式

```
<at user_id="ou_xxxxxxxxxxxxxxxx">Bot名称</at>
```
