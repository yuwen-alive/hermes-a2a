# 消息格式模板

## 任务分配

```
<at user_id="{open_id}">{名称}</at> 请执行：

**任务**：{任务描述}
**输入**：{需要的资源/数据}
**预期输出**：{期望的结果}
**截止**：{时间要求，可选}

[CONTINUE]
```

## 进度汇报

```
<at user_id="{open_id}">{名称}</at> 进展：

**已完成**：{完成的部分}
**进行中**：{正在做的部分}
**遇到问题**：{阻塞项，可选}

[CONTINUE]
```

## 任务完成

```
<at user_id="{open_id}">{名称}</at> 完成：

**结果**：{最终输出}
**产出物**：{文件路径/链接}
**备注**：{注意事项，可选}

[DONE]
```

## 信息通知

```
<at user_id="{open_id}">{名称}</at> 通知：

{通知内容}

[INFO]
```

## 求助升级

```
<at user_id="{open_id}">{名称}</at> 遇到问题：

**问题**：{问题描述}
**已尝试**：{已做的排查}
**需要**：{需要什么帮助}

[CONTINUE]
```

## 升级给人类

```
<at user_id="{用户open_id}">{用户名}</at> 需要介入：

**背景**：{简述上下文}
**问题**：{为什么需要人类}
**建议**：{可能的解决方案}

[NEEDS_HUMAN]
```
